Tìm kiếm phím tắt

Nhập từ khóa vào ô tìm kiếm để tìm theo tên chức năng, tổ hợp phím hoặc tên phần mềm.

VD: gõ "copy", "Ctrl", "debug"...

Lọc theo phần mềm / hệ điều hành

Dùng 2 dropdown bên phải thanh tìm kiếm để thu hẹp kết quả theo phần mềm cụ thể hoặc hệ điều hành.

Sắp xếp kết quả

Dùng dropdown Sắp xếp để hiển thị theo thứ tự A-Z theo phần mềm hoặc chức năng.

Thêm phím tắt tùy chỉnh

Nhấn nút ＋ ở góc dưới phải để thêm phím tắt của riêng bạn. Phím tắt tùy chỉnh được lưu trên trình duyệt và hiển thị nhãn ✦ Tùy chỉnh.

Phím tắt trùng lặp (cùng phần mềm + cùng tổ hợp phím) sẽ không được thêm.

Lưu trữ dữ liệu

Phím tắt mặc định được tải từ file data.json. Phím tắt tùy chỉnh lưu vào localStorage.
